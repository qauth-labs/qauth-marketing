# QAuth Landing Page

Marketing landing page for QAuth - the post-quantum ready authentication platform.

## Development

```bash
# From monorepo root
bun x nx dev qauth-landing

# Or using root scripts
bun run dev
```

## Build

```bash
bun x nx build qauth-landing
```

## Stack

- React 19 + TanStack Start
- TailwindCSS v4 + shadcn/ui
- Vite 7
- TypeScript 5.9
